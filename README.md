# django-user_a
Задание:
Создать проект со своей фамилией и в нем создать приложение со своим именем. В приложении сделать
3 страницы, на которые можно зайти по ссылкам
/ - ФИО, возраст, интересы (передать нужные параметры через kwargs)
about – откуда приехала, успеваемость в школе, любите/не любите учиться
contacts – указать свой GitHub и др. возможные варианты для связи (Telegram я ваш тоже у всех знаю, 
а значит и номер телефона)
